# vm-curator.org

Website for [vm-curator](https://github.com/mroboff/vm-curator) — a fast and friendly Rust TUI for managing QEMU/KVM virtual machines with GPU passthrough support.

## About

Static website built with pure HTML/CSS/JS. No frameworks, no build step.

## Hosting

Hosted on GitHub Pages at [vm-curator.org](https://vm-curator.org).

## Local Development

Open `index.html` in a browser, or serve locally:

```bash
python3 -m http.server 8080
```
